"Cons.xlsx" Data Dictionary

EventID - unique ID for each event, created by Qualtrics survey tool

StaffName - Panther ID of staff member who gave the consultation

StaffDept - department that the staff member giving consultation is associated with

UserStatus - status of patron(s) receiving the consultations; values may be either "UWM undergraduate student", "UWM graduate student", "UWM faculty or staff", "non-UWM person", or "don't know"

UserDept - if not a course-related consultation, the department to which the patron belongs; unknown represented as "NA"

Department - if course-related, the department of the course this consultation was about; unknown/no course represented as "NA"

Course - if course-related, the number of course this consultation was about; unknown/no course represented as "NA"

Faculty - if course-related, the faculty member for the course this consultation was about; unknown/no course represented as "NA"

Format - format of the consultation; values may be either "in person/face to face", "on the phone", "over e-mail", or "over snail mail"

ConsDate - date of the consultation; date format is YYYY-MM-DD

Length - length of the consultation in minutes

Count - how many patrons were in the consultation

Story - a touching story about this consultation; if no story is given, the field is marked "NA"

Notes - personal notes about the consultation; if no notes are input, this field is marked "NA"
